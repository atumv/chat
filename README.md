# Chat App

Simple chat app with React and Socket.io.

![](assets/preview.png)

## Install

```sh
git clone https://github.com/atumv/chat.git && cd chat/server && npm i && cd ../client && npm i
```

## Running

in one terminal window:
```sh
cd chat/server && npm start
```
in another:
```sh
cd chat/client && npm start
```
